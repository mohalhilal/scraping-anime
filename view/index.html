Anime Api - Endpoint

Endpoint Method Keterangan
/api/ongoing
GET
 Menampilkan anime Ongoing
/api/ongoing/:page
GET
 Menampilkan anime Ongoing berdasarkan Halaman.
Contoh : /api/ongoing/2
/api/complete
GET
 Menampilkan anime Complete / End
/api/complete/:page
GET
 Menampilkan anime Complete berdasarkan Halaman.
Contoh : /api/complete/2
/api/genre
GET
 Menampilkan daftar genre anime
/api/genre/:id
GET
 Menampilkan anime berdasarkan id genre.
Contoh : /api/genre/action
/api/genre/:id/:page
GET
 Menampilkan anime berdasarkan id genre dan halaman.
Contoh : /api/genre/action/2
/api/anime/:id
GET
 Menampilkan detail anime dan daftar episodenya.
Contoh : /api/anime/tomo-onanoko-sub-indo
/api/anime/:id
GET
 Menampilkan detail anime dan daftar episodenya.
Contoh : /api/anime/tomo-onanoko-sub-indo
/api/episode/:id
GET
 Menampilkan detail episode dalam anime seperti straming dan mirror streaming.
Contoh : /api/episode/tonanoko-episode-10-sub-indo
/api/mirror/:id
GET
 Menampilkan link mirror dengan id yang di dapat dari detail episode
Contoh : /api/mirror/eyJpZCI6MTMzNzgwLCJpIjowLCJxIjoiMzYwcCJ9
